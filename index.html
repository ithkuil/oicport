Scripto — The Colbert Report

Scripto is a a web-based system incorporating all of the major features of the previous Windows enterprise desktop application used to produce one of the most popular cable news/comedy programs, with the addition of collaborative script editing capabilities.

Technologies

Node.js, CoffeeScript, Etherpad-lite, EJS, JavaScript, MySQL, Socket.IO, bash, Ubuntu.

ADR Demo — JFORCES/NASA/Poulos Air & space

3D visualization for NASA ADR (Active Debris Removal) Program / JFORCES (Joint Forces Operational Readiness Combat Effectiveness System). Designed and implemented a system for visualizing satellite, debris, thrusters, sensors and data tables/graphs in browsers in 3D by distributing and processing graphics and data messages.

Technologies

Three.js, Node.js, CoffeeScript, Now.JS, some C

Cure — Node.js hosting service

I created a simple service to make it easy to deploy of virtual private servers preconfigured with Node.JS , MongoDB, redis etc. It is a server and DNS deployment and configuration using integrating RackSpace, Linode, and PayPal APIs

Technologies

Rackspace Cloud virtual private servers, Node.js, MongoDB, bash, JavaScript, etc.

Experiment — running AngularJS on the server

I was recently experimenting with running AngularJS (a component-based JavaScript framework) on the server with Node.js (in order to pre-render content for SEO and better page load times). It turned out to be pretty challenging. I documented my solution with a github project.

Technologies

AngularJS, Node.js, jsdom, CoffeeScript

Experiment — Loading Minecraft files in the browser

Minecraft is a popular game with simple block-based graphics. I have a for-fun project that reads the binary Minecraft data format and displays it in the browser in 3d.

Technologies

WebGL/Three.js, CoffeeScript, Node.js, HTML5
